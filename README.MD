# FaceToBMI

FaceToBMI 是一个基于人脸图像预测 BMI（身体质量指数）的 Python 项目。

## 特性

- 支持人脸检测与裁剪
- 基于深度学习模型预测 BMI

## 安装

1. 克隆本仓库：
    ```bash
    git clone https://github.com/yourusername/FaceToBMI.git
    cd FaceToBMI
    ```
2. 安装依赖：
    ```bash
    pip install -r requirements.txt
    ```

## 使用方法

```
python run.py
```

## 文件结构

- `run.py`：主程序入口
- `app`: FlaskApp框架
- `models/`：模型文件
- `requirements.txt`：依赖列表
